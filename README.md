## Prerequisites

- Java 17
- Maven

## Installation
Clone the repository:
```bash
git clone https://github.com/gitivanov/demo-springboot-rest.git
```
Navigate into the project directory:
```bash
cd demo-springboot-rest
```
Build the project:
```bash
mvn clean install
```
## Usage
To run the project, use the following command:
```bash
mvn spring-boot:run
```
